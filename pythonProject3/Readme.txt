--- Инструкция Запуска ---
1. После скачивание репозитория необходимо перейти в корневую папку проекта
2. Через терминал запустить команду: docker-compose up
3. После сборки и запуска контейнера перейти по отображенному ip - адресу
4. Для удобства отображения добавить к адресу эндпоинт /docs (документация апи, со всеми методами).
